# 网文核心技法参考

## 一、爽点设计

### 六种基础爽点模式

| 模式 | 结构 | 适用题材 | 密度建议 |
|------|------|---------|---------|
| 装逼打脸 | 轻视→展示实力→震惊 | 都市/玄幻/职场 | 每卷4-5次 |
| 扮猪吃虎 | 表面弱→关键时刻爆发→惊艳 | 重生/隐藏身份 | 每卷2-3次 |
| 越级反杀 | 实力悬殊→逆袭→不可置信 | 玄幻/武侠/竞技 | 每卷1-2次 |
| 打脸权威 | 权威质疑→实力证明→认可/尴尬 | 职场/学院 | 每卷2-3次 |
| 反派翻车 | 反派得意→计划破产→狼狈 | 全题材 | 每卷3-4次 |
| 情感升温 | 平淡→意外惊喜→情感推进 | 言情/日常 | 按主线需要 |

### 爽点三阶布局

**微爽点（次/章）**：
- 一句妙语、一个反转信息、一个小成功
- 每章至少1个，分散在行文中

**中爽点（次/卷）**：
- 击败一个对手、获得一个新能力、揭开一个秘密
- 每卷5-8个，均匀分布

**大爽点（次/全书）**：
- 终极对决、身份揭晓、情感高潮
- 全书3-5个，分别在三卷中

### 爽点黄金公式

```
铺垫（制造期待）→ 展示（主角行动）→ 反馈（周围反应）→ 余韵（主角整理收获）
```

四段缺一不可。跳过"反馈"阶段会让爽点打五折——读者需要看到别人被震撼、后悔、羡慕。

## 二、章节结构

### 标准章节模板

```
┌─────────────────────────────────┐
│ 前200字：钩子（冲突/悬念/异常） │  ← 抓读者
├─────────────────────────────────┤
│ 进展（推进主线/支线）           │  ← 讲故事
├─────────────────────────────────┤
│ 微爽点（小反转/信息差/幽默）     │  ← 保持阅读愉悦感
├─────────────────────────────────┤
│ 发展（持续推进）                │
├─────────────────────────────────┤
│ 微爽点（或深化矛盾）            │
├─────────────────────────────────┤
│ 后200字：钩子（未闭合问题）      │  ← 留读者
└─────────────────────────────────┘
```

### 钩子类型轮换

避免连续3章用同类型钩子。推荐轮换模式：

```
悬念钩 → 冲突钩 → 异常钩 → 动作钩 → 对话钩 → 悬念钩 ...
```

## 三、对话技巧

### 对话四原则

1. **每段对话有胜负** — 不是"交流信息"，是"争夺优势"
2. **少说多演** — 用动作引导对话，少用"XX说/道"
3. **不要回答全部问题** — 留白让读者自己猜
4. **节奏变化** — 紧张时短句快速切换，放松时长句

### 对话引导格式

```
❌ 通用型（呆板）：
"你来了。"他说。
"嗯。"她点点头。
"路上顺利吗？"
"还行。"

✅ 动作引导型（生动）：
他抬起头看见她站在门口，手里的烟灰落了一截。
"你来了。"
她没进门，靠着门框，手插在兜里。
"嗯。"
"路上顺利吗？"
她没回答，只是把另一只手里的东西扔了过来——他用单手接住了。
```

### 意图冲突模板

```
A想要____，所以要劝说/阻止/欺骗B
B想要____，所以反抗/假装/利用A
__________________________________
对话的核心张力来自这两个目标的碰撞
```

**无效对话**：A问路，B指路（纯信息传递）
**有效对话**：A想借道，B不肯（利益冲突）
**更有效**：A想借道，B假装不肯但实际在试探A（双重意图）

## 四、节奏控制

### 快节奏技法
- 短句短段（每段1-3句）
- 动作连续（A做→B做→A再做）
- 少描写多行动
- 对话推进不解释

### 慢节奏技法
- 长句（20-40字）
- 环境描写穿插
- 内心独白
- 细节放大（一个动作写3个感官）

### 节奏曲线（每章）

```
       高潮
       /\
      /  \
     /    \
    /      \
   /        \
  /          \
 /            \
/              \
起势        回落→钩子
```

前200字入题 → 逐步加码 → 中段小高潮 → 回落 → 尾段新钩子

## 五、环境与氛围

### 灵异题材氛围技法
- 感官锚定：触觉（凉/湿/粘）> 听觉（吱呀/滴水/脚步）> 视觉（模糊/光影）
- 反常即恐怖：日常事物被打破规则（灯该亮不亮、该安静时有声音）
- 信息遮蔽：读者和主角一样不知道发生了什么
- 节奏控制：慢→慢→突然快（积累紧张后爆发）

### 环境描写的经济原则
- 用1-2个细节代替整段描写
- 环境通过角色的感受来写（过滤视角）
- 不重复使用同一意象

## 六、人物塑造

### 弧光设计四象限

| 象限 | 定义 | 示例 |
|------|------|------|
| 向上 | 从弱到强、从怯到勇 | 主角成长 |
| 向下 | 从强到弱、从信到疑 | 导师之死 |
| 转折 | 信念崩塌到重建 | 复仇者的宽恕 |
| 坚守 | 始终不变的核心品质 | 诚实、善良 |

每个主要角色选择1个象限。

### 配角的"三不"原则
- **不用**配角只说"是/好/对"（每句对话都要有信息或态度）
- **不让**配角的功能只服务于主角（给他独立的支线目标）
- **不把**配角当"信息发布器"（背景故事用行动揭示，不是对话说教）
